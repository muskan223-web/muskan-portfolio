Place font files here:

Download Clash Display font from:
https://www.fontshare.com/fonts/clash-display

Place these files in this folder (public/fonts/):
- ClashDisplay-Regular.woff2
- ClashDisplay-Medium.woff2
- ClashDisplay-Semibold.woff2
- ClashDisplay-Bold.woff2

The website will use Inter (Google Fonts) as body font and 
Clash Display as heading font automatically.

If you don't place these files, the website will fall back to 
system fonts which still looks clean.
