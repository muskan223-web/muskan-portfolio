## STEP 1 — Open a Terminal in the Project Folder

1. Open **Windows Explorer**
2. Navigate to: `C:\Users\kumar\OneDrive\Desktop\project`
3. Right-click inside the folder → "Open in Terminal" (or PowerShell)

---

## STEP 2 — Install Dependencies

In the terminal, run:

```
npm install
```

Wait until it completes (2–3 minutes).

---

## STEP 3 — Add Your Images

Copy your photos into the `public/images/` folder:

| Your Photo | Rename it to |
|-----------|-------------|
| First image (selfie portrait) | `hero-portrait.jpg` |
| Second image (full body) | `bg-silhouette.jpg` |

---

## STEP 4 — Launch the Website

```
npm run dev
```

Then open your browser and go to: **http://localhost:3000**
